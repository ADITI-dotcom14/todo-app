## 📌 Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Data stored using LocalStorage (persists after refresh)

* HTML
* CSS
* JavaScript

## 📦 Installation


## 💡 How It Works

This app stores tasks in the browser using LocalStorage.
Each task can be marked complete or deleted, and changes are saved automatically.
